The exercises were solved by having two projects, 
Backend with Restsharp and Frontend with Selenium and C#

Pre-requisites:
Have dotnet installed on your machine (https://dotnet.microsoft.com/download))
Have Chrome browser installed on your machine

Steps to run the tests:
1. Go to the TechProject folder, open a command prompt and run the following commands:
2. dotnet build
3. dotnet test
